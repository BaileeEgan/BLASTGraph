# Installation

```
conda create -n blastgraph --file requirements.txt
```


## Running example data:

```
python blastgraph.py -f example_data/queries/F3D0_S188_L001_R1.fasta -d example_data/databases/16S_ribosomal_RNA -e 1e-10 
--force
```
